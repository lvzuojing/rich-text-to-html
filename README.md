# 📝 富文本转 HTML 源码工具

基于 [TinyMCE](https://www.tiny.cloud/) 的在线富文本排版与 HTML 源码提取工具。打开即用，无需安装。

## ✨ 功能特性

- **Word 级排版**：加粗、斜体、下划线、删除线、字体、字号、颜色、对齐、缩进、列表等
- **表格支持**：插入和编辑表格，完美保留从 Word/Excel 粘贴的表格样式
- **一键提取**：排版完成后，点击按钮即可将 HTML 源码复制到剪贴板
- **粘贴保留格式**：从 Word、Excel、网页复制内容粘贴，格式完整保留
- **中文界面**：内置中文语言包，操作无障碍
- **纯前端**：单个 HTML 文件，无需后端，无需构建，浏览器打开即用

## 🚀 使用方法

1. 下载 [`富文本转 HTML 源码工具.html`](./富文本转%20HTML%20源码工具.html)
2. 用浏览器打开
3. 在编辑器中排版内容（支持直接粘贴 Word/Excel/网页内容）
4. 点击底部 **「排版完毕，一键提取 HTML 代码」** 按钮
5. HTML 源码已复制到剪贴板，粘贴到需要的地方即可

## 🛠 技术栈

- [TinyMCE 6.8.2](https://www.tiny.cloud/) — 富文本编辑器引擎
- 原生 HTML/CSS/JS，零依赖构建
